about=The MRF Translado GNSS plugin automates coordinate adjustment processes according to Brazilian INCRA/SIGEF standards. It allows GNSS data import, coordinate translation using PPP IBGE or SIGEF Memorial as reference, layer creation in QGIS, and technical report generation in PDF format. This version includes optional variance propagation, automatic disabling of variance propagation for SIGEF Memorial, explanatory help for variance propagation, and TXT template export for standardized data input. This plugin requires external Python libraries such as pandas and pdfplumber. Installation instructions are available in the repository.
